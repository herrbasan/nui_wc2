Component API

Programmatic — nui.components.dropzone.create(zones, callback, target)

Parameter Type Description
zones Array Array of { name, label } objects. name becomes the data-drop value, label is the display text (supports HTML).
callback Function Called on drop with (detail, event). detail contains { zone, dataTransfer, originalEvent }.
target HTMLElement Container to append the dropzone to. Must have position: relative and a defined height.

Attributes

Attribute Description
active Present when the overlay is visible (managed automatically by drag events).

Events

Event Name Detail Description
nui-dropzone-open — Fired when the overlay becomes visible (files dragged over container).
nui-dropzone-drop { zone, dataTransfer, originalEvent } Fired when files are dropped. zone is the data-drop value.
nui-dropzone-close — Fired when the overlay hides after a drop or drag-leave.
