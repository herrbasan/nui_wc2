Progress

Provides visual feedback on the progress of a task, or indicates busy states. Optionally shows a percentage label.

Linear Progress Bar

The standard progress bar automatically displays numerical percentage based on its value and maximum max attribute (default 100). You can hide it using the hide-text attribute.

Circular Progress

Set type="circular" for a spinner-like progress indicator. Similar to linear bars, text can be disabled using hide-text.

Indeterminate (Busy)

Set type="busy" when the specific value is unknown or it implies an ongoing activity.

Circular Indeterminate (Busy)

Set type="circular-busy" for a compact, button-sized infinite spinner.

Custom Sizing & Alignment

The size attribute accepts CSS dimension formats (e.g. 4rem, 30px, 100%). For circular variants, it adjusts the diameter. For bars, it controls the thickness track height.

Because circular variants use an SVG with 100% width/height internally, using size="100%" allows them to cleanly fill and center inside arbitrary container wrappers.
