Interactive / Clickable Area

Add the interactive attribute to a card. If the card contains an <a class="nui-card-link">, the link's hit area will seamlessly cover the whole card, meaning you avoid wrapping complex nested elements in a large anchor tag. If no link is present, you can add a data-action attribute directly to the card. Keyboard navigation (pressing Enter or Space) is handled automatically.

Flip Cards

Add layout="flip" and configure internal .nui-card-front and .nui-card-back elements. It flips automatically on pressing Space/Enter or via a data-action="card-flip" button.
