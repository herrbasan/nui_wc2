Card Component

A flexible container format for structured content, media, and actions.

Standard Layout

The standard card stacks its content vertically. It automatically provides consistent borders, rounded corners, and subtle shadows using our built-in CSS variables.

Card Title

This is a basic vertical card handling image, text, and actions gracefully.

Horizontal Layout

Use layout="horizontal" to position media and content side-by-side. Useful for lists or wide displays.

Horizontal Strategy

Content is arranged flexibly when oriented left-to-right. Perfect for responsive lists.

Image-only with Text Overlay

By specifying layout="image-only", the card becomes an image container with layered text on top.

Overlay Feature

Description sitting on top of the image.

Expandable Clickable Area (Interactive)

Add the interactive attribute to a card. If the card contains an <a class="nui-card-link">, the link's hit area will safely cover the entire card without making the developer wrap nested elements in a large anchor tag. If no link is present, pressing Enter/Space handles a synthetic click event.

Interactive Link Title

Hovering reveals a deeper shadow. Clicking outside the text still routes the link.

Flip Cards

Add layout="flip" and configure internal .nui-card-front and .nui-card-back elements. It flips automatically on pressing Space/Enter or via a data-action="card-flip" button.
