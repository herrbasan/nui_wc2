Component API

The tabs component exposes properties and methods for programmatic control.

Methods

Method Description
selectTab(idOrIndex) Activates a tab by panel ID (string) or zero-based index. Expands parent groups if necessary.

Events

Event Name Description
nui-tab-change Fired when the active tab changes. e.detail.tab contains the activated button element, and e.detail.panel contains the associated panel element.
