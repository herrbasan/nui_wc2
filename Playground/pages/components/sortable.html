Component API

The sortable component exposes methods for programmatic control and events for monitoring changes.

Methods

Method Description
getItems() Returns an array of objects with { id, element } for each sortable item.
addItem(html) Appends a new item via HTML string. The string should contain a <nui-sortable-item> element.
setItems(array) Replaces all items with an array of HTML strings.
clear() Removes all items from the sortable list.

Events

Event Name Description
nui-sortable-change Fired when the order of items changes. e.detail.order contains an array of data-id strings in the new order.
