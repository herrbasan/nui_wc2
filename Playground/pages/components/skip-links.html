Skip Links

Skip links provide keyboard users with a way to quickly jump to important page sections, improving navigation efficiency and accessibility. The nui-skip-links component makes it easy to add skip links to your application.

Features

  • Automatic Detection - Auto-generates links for common NUI app structure (top-nav, side-nav, main content)
  • Declarative HTML - Define custom skip links with simple anchor tags
  • Keyboard Friendly - Links are hidden until focused via Tab key
  • Smart Focus Management - Automatically adds tabindex="-1" to targets for proper focus handling
  • Smooth Scrolling - Animated scroll to target section

Automatic Mode (Recommended)

Simply add <nui-skip-links></nui-skip-links> to your app and it will automatically detect and create skip links for:

  • nui-top-nav - Skip to navigation
  • nui-side-nav - Skip to sidebar
  • main - Skip to main content
<nui-app>
	<!-- Auto-generates skip links for nui-app structure -->
	<nui-skip-links></nui-skip-links>
	
	<nui-top-nav>...</nui-top-nav>
	<nui-side-nav>...</nui-side-nav>
	<nui-content>
		<main>...</main>
	</nui-content>
</nui-app>

Declarative Mode

Define custom skip links by providing anchor tags inside the component:

<nui-skip-links>
	<a href="#main-content">Skip to main content</a>
	<a href="#search">Skip to search</a>
	<a href="#footer">Skip to footer</a>
</nui-skip-links>

<header>...</header>
<main id="main-content">...</main>
<div id="search">...</div>
<footer id="footer">...</footer>

How It Works

  1. Hidden by default - Skip links are positioned off-screen until focused
  2. Tab to reveal - Press Tab from the top of the page to reveal the first skip link
  3. Focus cascade - When one link is focused, others become visible inline
  4. Auto-setup - Component ensures target elements have proper id and tabindex="-1" attributes

Usage Tips

Placement

Place nui-skip-links as the first focusable element in your document, typically right after the opening <body> or inside <nui-app>.

Testing

To test skip links:

  1. Reload the page
  2. Press Tab - the first skip link should appear
  3. Press Enter to activate
  4. Focus should move to the target section

Accessibility Notes

  • Skip links are required for WCAG 2.1 Level A compliance
  • They help users with mobility impairments bypass repetitive navigation
  • Screen readers announce skip links as navigation landmarks
  • The component automatically adds role="navigation" and aria-label="Skip links"

Live Example

This playground application uses nui-skip-links in automatic mode. Try it yourself:

  1. Click in the browser address bar (or press Ctrl+L / Cmd+L)
  2. Press Tab repeatedly to see the skip links appear at the top of the page
  3. Press Enter on any skip link to jump to that section
Tip: Skip links are especially useful on pages with complex navigation. They allow keyboard users to quickly reach the main content without tabbing through dozens of navigation items.

Component Reference

Component: nui-skip-links

Mode Description
Automatic Empty component auto-generates skip links for nui-app structure
Declarative Define custom skip links using anchor tags as children

Automatic Targets

Element Label Condition
nui-top-nav "Skip to navigation" If present in nui-app
nui-side-nav "Skip to sidebar" If present in nui-app
main "Skip to main content" If present in document

ARIA Attributes (Auto-applied)

Attribute Value Target
role "navigation" nui-skip-links element
aria-label "Skip links" nui-skip-links element
tabindex "-1" Target elements (if missing)
id Generated or existing Target elements (if missing)
