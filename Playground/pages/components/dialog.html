LLM Guide - nui-dialog

Dialog Component

Modal and non-modal dialogs for user interactions requiring a response

Overview

The <nui-dialog> component wraps the native HTML <dialog> element with animations, placement options, and accessibility enhancements. Use dialogs when you need user input or confirmation—for passive notifications, use banners instead.

Key features:

  • Native dialog element with backdrop
  • Placement options (top, center, bottom)
  • Scale animations for open/close
  • System dialogs (alert, confirm, prompt)
  • Click-outside-to-close behavior
  • Escape key handling

Page Mode

	
Add mode="page" to a dialog to create a layout optimized for large structures like settings menus or forms. Page mode dialogs use CSS grid/flexbox to maximize use of the viewport while keeping the header and footer fixed and making the main content area scrollable.

	
	

		Virtual Scroller Boundaries: If embedding a full-height virtual scroller like <nui-list> that physically dictates its own inner layout measurements based on viewport calculations, add the additional content-scroll="false" parameter to the <nui-dialog>. This prevents the dialog's default content wrapper from pushing the list down infinitely causing it to collapse.

Component API

		
The dialog system provides factory methods on nui.components.dialog and events on <nui-dialog> elements.


		
Factory Methods (Preferred)

		

			
				

					Method
					Description
				

			
			
				

					alert(title, message, options?)
					Simple alert dialog. Returns Promise with button value.
				

				

					confirm(title, message, options?)
					Confirmation dialog. Returns boolean Promise.
				

				

					prompt(title, message, options?)
					Form prompt with fields. Returns object or null.
				

				

					page(title, content, options?)
					Complex page-style dialog. Returns {dialog, main, result}.
				

			
		


		
Events

		

			
				

					Event
					Description
				

			
			
				

					nui-dialog-open
					Fired when dialog becomes visible.
				

				

					nui-dialog-close
					Fired on close. e.detail.returnValue contains the result.
