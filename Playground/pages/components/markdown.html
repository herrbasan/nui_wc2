Markdown (Addon)

A lightweight component and utility for rendering Markdown into HTML.

Basic Usage

Wrap your markdown in a <script type="text/markdown"> element inside the nui-markdown component. Remember to escape any </script> tags in your content as <\/script>.

Supported Syntax

The parser supports headers, links, emphasis, lists, separators, simple tables, blockquotes, and code blocks using nui-code (which requires the nui-syntax-highlight module for colored syntax in fenced code blocks).

Streaming & Incremental Rendering

For AI & LLM applications, ui-markdown` includes built-in block-level incremental rendering. Instead of re-parsing and replacing the entire DOM (which causes severe UI flickering and thrashing, especially for syntax-highlighting code blocks), you can stream chunks to the component. The DOM is kept completely stable for sealed blocks, creating a butter-smooth streaming experience.

Escaping Script Tags

		
When your markdown contains code examples with </script> tags, you must escape them as <\/script>. HTML parsers treat </script> anywhere inside a script element as closing it, even inside code examples.

		
		

			Always escape </script> as <\/script> in your markdown content to prevent the HTML parser from closing the script element early.

Why This Happens

		
The HTML parser scans for </script> to find the end of a script element. It doesn't understand that the tag is inside a code example - it just sees the string and closes the element. Escaping with <\/script> prevents this while still displaying correctly in the rendered output.
