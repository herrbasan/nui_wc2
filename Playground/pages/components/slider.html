LLM Guide - nui-slider

Slider

Overview

The <nui-slider> component wraps a native <input type="range"> with a custom visual design. It uses nui.util.enableDrag for smooth touch and mouse interaction with a large hit area but slim visual appearance.

Basic Usage

Wrap a native range input. The component automatically creates the visual track, fill, and thumb.

Value: 50

Custom Range

Set min, max, and step on the native input.

Value: 5 / 10

Available Events

Listen to native input (continuous) and change (on release) events.

The enableDrag Utility

The slider component uses nui.util.enableDrag internally. You can use this utility directly for custom drag interactions like video scrubbers or 2D controls.

Component API

The slider component exposes methods for programmatic control and uses native input events.

Methods

Method Description
getValue() Returns the current value of the slider (same as accessing the inner input's value property).
setValue(val) Sets the slider value programmatically.

Events

Events are fired by the inner native <input> element and bubble up to the <nui-slider> wrapper.

Event Description
input Fired continuously while dragging (like native range input).
change Fired when the user releases the slider (like native range input).
