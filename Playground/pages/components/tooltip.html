Tooltip Component

Universal tooltip for annotating UI elements on hover or focus.

Basic Usage (Auto Placement)

A tooltip automatically attaches to the element immediately preceding it. By default, it uses auto-placement based on the available screen space. It uses the native Popover API for accessible, top-layer rendering without complex JS scroll tracking or z-index issues.

This is an auto-placed tooltip, resolving space dynamically.

Rich Content & Interactivity

Tooltips can hold fully formatted rich content like headings, paragraphs, and interactive buttons without closing when you hover over them, closing smoothly after a short delay so the mouse can travel to it.

Tooltip Title

Tooltips display informative text when users hover over, focus on, or tap an element.

Using the 'for' attribute

For more complex layouts, you can explicitly link a tooltip to an element using the for attribute matching the target's id. This also pairs them correctly for accessibility via aria-describedby.

Placement Options

While auto mode is recommended, you can use the position attribute to manually choose where the tooltip appears. Supports top, bottom, left, and right.
