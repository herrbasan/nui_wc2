LLM Guide - nui-tag-input

Tag Input

A reusable tag management component for lists of removable items. Used standalone or composed by multi-select components.

Pre-populated Tags

Initialize with tags using hidden inputs. These will be parsed on component upgrade.

Try It

Tags: active, published, featured
Keyboard:
  • ← / → — Move between tags
  • Enter / Space on tag — Remove tag
  • Home — Jump to first tag
  • End — Jump to last tag (or input if editable)
  • Escape — Clear focus

Basic Usage

Tags are managed programmatically via the component's API. Hidden inputs provide form compatibility.

Component API

The tag input component exposes methods for programmatic control and events for monitoring changes.

Methods

Method Description
addTag(value, label?) Adds a new tag. Returns true if successful, false if tag already exists. Optionally provide a label for display.
removeTag(value) Removes a tag by its value. Returns true if removed, false if not found.
getValues() Returns an array of tag values (strings).
listTags() Returns an array of tag objects with value and label properties.
hasTag(value) Returns true if a tag with the given value exists.
clear() Removes all tags.

Events

Event Name Description
nui-tag-add Fired when a tag is added. e.detail.value contains the value and e.detail.label contains the label.
nui-tag-remove Fired when a tag is removed. e.detail.value contains the removed value.
nui-change Fired when tags change. e.detail.values contains an array of all current values.
