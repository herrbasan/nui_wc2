Interactive Example

Standalone Badges

Use the <nui-badge> element to add inline labels or indicators to an interface.

Integrated Status Badge

Instead of wrapping elements in complex DOM hierarchies to show a notification dot or count, you can simply add the data-badge attribute to any container element. The badge uses a pseudo-element, enabling a zero-bloat approach for notification indicators (e.g. on icons or buttons).

An empty string (data-badge="") renders as a small indicator dot, ideal for subtle unread notifications.
