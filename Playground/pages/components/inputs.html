Input Components

Form input components with native accessibility, validation, and custom styling.

Text Input

Basic text input with the <nui-input> component. Works standalone or wrapped in <nui-input-group> for labels.

We'll never share your email with anyone.

Clearable Input

Add the clearable attribute to show a clear button when the input has a value.

Standalone Input

Inputs work without a group wrapper.

Input Types

Standard HTML input types are supported.

Disabled & Read-only

Standard HTML attributes for disabled and read-only states.

Textarea

Multiline text input with <nui-textarea>. Supports auto-resize and character counting.

Auto-resize

Add auto-resize to make the textarea grow with content.

Character Count

Add show-count to display character count. Works with maxlength.

Checkbox

Custom-styled checkboxes with <nui-checkbox>. Native accessibility preserved.

Checkbox Group

Use <nui-input-group> to create a labeled group of checkboxes.

Radio Buttons

Custom-styled radio buttons with <nui-radio>.

Disabled Radio

Radio buttons can be disabled individually.

Validation

Inputs use native constraint validation. Validation runs on blur and shows error messages automatically.

Form Row

Use <nui-form-row> to place inputs, checkboxes, and buttons side by side at uniform height.
