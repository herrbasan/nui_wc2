Icon Component

SVG sprite-based icon system for consistent, scalable icons

Overview

The <nui-icon> component uses an SVG sprite system for efficient icon rendering. All icons are loaded from a single sprite file (material-icons-sprite.svg) which reduces HTTP requests and allows for instant icon rendering.

Features

  • SVG Sprite System - Single file contains all icons, efficient loading
  • Scalable - Vector-based, scales perfectly at any size
  • CSS Themeable - Icons inherit color from CSS, easy to customize
  • Material Design Icons - Comprehensive set of modern, clean icons
  • Inline Usage - Can be used inline with text

Basic Usage

Use the name attribute to specify which icon to display:

<nui-icon name="settings"></nui-icon>
<nui-icon name="person"></nui-icon>
<nui-icon name="search"></nui-icon>

Sizing

Icons inherit font-size from their container or can be sized directly with CSS:

<nui-icon name="notifications" style="font-size: 1rem;"></nui-icon>
<nui-icon name="notifications" style="font-size: 1.5rem;"></nui-icon>
<nui-icon name="notifications" style="font-size: 2rem;"></nui-icon>
<nui-icon name="notifications" style="font-size: 3rem;"></nui-icon>

Coloring

Icons inherit the current text color and can be styled with CSS:

<nui-icon name="security" style="color: red;"></nui-icon>
<nui-icon name="analytics" style="color: orange;"></nui-icon>
<nui-icon name="done" style="color: green;"></nui-icon>
<nui-icon name="info" style="color: blue;"></nui-icon>

Inline with Text

Icons can be used inline with text and will align with the baseline:

<p>
	<nui-icon name="mail"></nui-icon> Email Settings
</p>
<p>
	<nui-icon name="person"></nui-icon> User Profile
</p>

Implementation Details

SVG Sprite System

All icons are stored in a single SVG sprite file with symbol definitions:

<svg xmlns="http://www.w3.org/2000/svg">
	<symbol id="icon-search" viewBox="0 0 24 24">
		<path d="M15.5 14h-.79l-.28-.27A6.471 6.471 0 0 0 16 9.5 6.5 6.5 0 1 0 9.5 16c1.61 0 3.09-.59 4.23-1.57l.27.28v.79l5 4.99L20.49 19l-4.99-5zm-6 0C7.01 14 5 11.99 5 9.5S7.01 5 9.5 5 14 7.01 14 9.5 11.99 14 9.5 14z"/>
	</symbol>
</svg>

Component Usage

The component references sprite symbols using SVG <use> element:

<nui-icon name="search">
	<svg>
		<use href="material-icons-sprite.svg#icon-search"></use>
	</svg>
</nui-icon>

Benefits

  • Single HTTP Request - All icons loaded once, cached by browser
  • No Font Files - True vector graphics, not font characters
  • Instant Rendering - No FOUC (Flash of Unstyled Content)
  • Easy Updates - Regenerate sprite to add/update icons
  • CSS Control - Full styling control via CSS

Sprite Generation

The sprite is generated from individual SVG files using generate_icon_sprite.py. This allows easy addition of new icons - just add SVG files and regenerate the sprite. 

python assets/generate_icon_sprite.py
