Overlay

A raw modal container for lightboxes, full-screen loaders, and custom modals.

Declarative Usage (Default)

The <nui-overlay> component acts as a generic container that leverages the native <dialog> API to present content centrally over a shaded backdrop. Unlike <nui-dialog>, it doesn't enforce any padding or borders.

Programmatic Usage (Advanced / Async)

Use the programmatic API when you need to control the overlay during asynchronous workflows, such as showing a loader while saving data.

By adding the blocking attribute, the overlay will ignore backdrop clicks and Escape key presses. This is ideal for loading indicators.

Available Events

Listen for these custom events to react when the overlay opens or closes.

Component API

The overlay component exposes methods for programmatic control and events for monitoring state changes.

Methods

Method Description
showModal() Opens the overlay as a modal dialog with backdrop.
close(returnValue?) Closes the overlay. Optionally provide a return value to pass to event handlers.

Events

Event Name Description
nui-overlay-open Fired when the overlay becomes visible.
nui-overlay-close Fired when the overlay closes. e.detail.returnValue contains any value passed to close().
nui-overlay-cancel Fired when the overlay is cancelled (e.g., by pressing Escape or clicking backdrop).
