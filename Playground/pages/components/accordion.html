Accordion Component

Expandable content sections using <nui-accordion>. Built on native <details> elements.

Basic Usage

Wraps standard HTML <details> elements. Provides consistent styling and optional behavior enhancements.

Section 1: Introduction

Welcome to the NUI Accordion. This component is designed to handle various types of content, from simple text to complex HTML structures.

It leverages the native <details> and <summary> elements, ensuring that the content is accessible and semantic. The animation you see is a progressive enhancement, meaning the component works perfectly fine even if JavaScript is disabled or fails to load.

Section 2: Features

The NUI Accordion comes with several built-in features:

  • Semantic HTML: Built on standard web platform elements.
  • Accessibility: Fully accessible via keyboard and screen readers.
  • Animation: Smooth height transitions (enabled by default).
  • Styling: Easily customizable via CSS variables.

You can nest other components inside these panels as well.

Section 3: Conclusion

In conclusion, the accordion is a versatile tool for organizing content. By hiding sections that aren't immediately relevant, you can reduce cognitive load and make your pages easier to scan.

Remember to use the exclusive attribute if you want to enforce a "one-at-a-time" policy, which is often useful for FAQs or step-by-step instructions.

 
<nui-accordion>
  <details>
    <summary>Section 1</summary>
    <div>...</div>
  </details>
  <details>
    <summary>Section 2</summary>
    <div>...</div>
  </details>
</nui-accordion>

Exclusive Mode

Add the exclusive attribute to allow only one section to be open at a time.

First Option

This is the First Option. In exclusive mode, opening another section will automatically close this one.

This behavior is particularly useful for:

  • Reducing visual clutter on the page.
  • Guiding the user's focus to a single topic.
  • Creating a "wizard-lite" experience where users step through information.
Second Option

Here is the Second Option. Notice how the first one closed smoothly when you opened this?

The animation logic handles the coordination between the opening of this element and the closing of the previous one, ensuring a jank-free experience.

Third Option

And finally, the Third Option.

Exclusive mode is implemented by listening to the toggle event (or click events when animated) and managing the open state of sibling details elements.

 
<nui-accordion exclusive>
  <details open>...</details>
  <details>...</details>
</nui-accordion>

Visual Variants

Accordions can be styled to fit different contexts.

Clean Variant

Use the nui-accordion-clean class for a minimalist look without borders and backgrounds.

Clean Style Item

This variant removes the borders and background colors for a cleaner, more minimalist look. It fits well within other containers or cards.

Another Clean Item

The clean variant maintains the same animated behavior as the default style, just with different visual treatment.

No Animation

Add the no-animation attribute to disable animations and use native browser behavior.
