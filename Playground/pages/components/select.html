LLM Guide - nui-select

Select Component

Enhanced select elements with search, multi-select, and accessible custom UI

Overview

The <nui-select> component enhances native <select> elements with a custom UI, search capabilities, and multi-select tag management, while maintaining full accessibility and form compatibility.

Declarative Usage (Default)

A simple dropdown for selecting a single option. The first empty-value option serves as placeholder text. You can toggle its state declaratively using data-action="select-open" or select-close.

Tip: Use syncOptions() when you've manually modified the native <select> element via DOM manipulation. Use setItems() to programmatically replace all items. refresh() is kept as an alias for backward compatibility.

Async Data Loading Pattern

Common pattern: disable the select while loading data, then populate options and enable when ready. The select can be updated at any point—add options, change selection, query state—all synchronously once the component is defined.

Key Points:

  • Use loadOptions() for the simplest async pattern—it handles all loading states
  • Use showLoading()/hideLoading() for manual control
  • Use isLoading() to query current state
  • Use disable()/enable() or setDisabled(bool) to control availability
  • Query state anytime with getValue(), isDisabled(), isOpen(), getItems()

Keyboard Navigation

The select component provides comprehensive keyboard accessibility following WAI-ARIA best practices.

When Closed (Control Button Focused)

Key Action
Enter / Space Open popup. Focus search input (searchable) or first option
ArrowDown Open popup and focus first option (or search input)
ArrowUp Open popup and focus last option (or search input)
Home Open popup and focus first option (non-searchable)
End Open popup and focus last option (non-searchable)
A-Z, 0-9 Open popup. Focus search input (searchable) or jump to option
Tab Move focus to next/previous field

Search Input (Searchable Selects)

Key Action
ArrowDown / Tab Move focus to first option
ArrowUp Move focus to last option
Escape Close popup, return focus to control

When Open (Options List Focused)

Key Action
ArrowDown Move focus to next option
ArrowUp Move focus to previous option (or search input if at top)
Home Move focus to first option
End Move focus to last option
Enter / Space Select focused option and close (single) or toggle (multi)
Escape Close popup without selecting, return focus to control
Tab Close popup and move to next field
A-Z, 0-9 Type-ahead search to jump to matching options
Accessibility: The select component uses roving tabindex for option navigation, provides visual focus indicators, and maintains ARIA compliance for screen reader compatibility.

Attributes

Attribute Type Description
searchable boolean Enable search/filter functionality
mobile-sheet boolean Force bottom sheet on all viewports (auto-triggers ≤ 640px)
placeholder string Placeholder text (also set via empty option)
label string Accessibility label (auto-detected from parent label)
