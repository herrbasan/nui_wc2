Markdown (Addon)

A lightweight component and utility for rendering Markdown into HTML.

Basic Usage

Import the module and wrap your markdown in a <script type="text/markdown"> element inside the nui-markdown component.

Supported Syntax

The parser supports headers, links, emphasis, lists, separators, simple tables, blockquotes, and code blocks using nui-code.
