Rich-Text Editor

A lightweight, dependency-free rich text editor utilizing native browser APIs.

Experimental: This component is currently in an experimental phase. Features, API, and styling are subject to change.

Basic Usage

Use the <nui-rich-text> element to provide an accessible WYSIWYG editing experience. It leverages existing NUI components for its toolbar.
