Context Menu

A lightweight, keyboard-accessible context menu popup triggered via right-click or programmatically. Shares visual styling with nui-menu.

Overview

The nui-context-menu module provides a floating popup menu similar to native browser context menus or application menus. It's programmatic-only (not a declarative custom element).

Key features:

  • Programmatic API - create and show menus via JavaScript
  • Submenu support with hover delay (200ms)
  • Full keyboard navigation (arrows, Enter, Escape)
  • Auto-flip positioning to stay within viewport
  • Single global instance - opening closes any existing menu
  • Shared styling with nui-menu dropdowns

Keyboard Navigation

Keyboard Shortcuts:

  • Arrow Up/Down: Navigate between items
  • Enter: Activate focused item (opens submenu if present)
  • Escape: Close menu
  • Arrow Right: Open submenu (when focused on item with submenu)
  • Arrow Left: Close submenu (when in submenu) or close menu (when at root)

API Reference

Factory Function

Signature Description
contextMenu(items, options) Creates a new context menu instance with the given items array and options.

Options

Property Type Description
onAction (action: string, item: object) => void Callback when an item with an action is clicked/activated.
onSubmenuOpen (label: string) => void Callback when a submenu opens.

Menu Item Properties

Property Type Description
label string Display text for the menu item.
action string Identifier passed to onAction callback when item is activated.
items array Submenu items array (creates nested submenu).
type 'separator' Renders a horizontal divider.
disabled boolean Prevents interaction when true.

Instance Methods

Method Description
show(x, y, triggerEl?) Display menu at viewport coordinates. Optional trigger element enables scroll detection.
showAt(element) Display menu aligned to an element's position.
hide() Close the menu if open.
destroy() Close and clean up the menu instance.

Static Methods

Method Description
contextMenu.closeAll() Close any open context menu.
