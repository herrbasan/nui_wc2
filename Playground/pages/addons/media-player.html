LLM Guide - nui-media-player

Media Player

An experimental and customizable video and audio player component built on top of native media elements.

Experimental: This component is currently in an experimental phase. Features, API, accessibility and styling are subject to change.

Video Player

Wrap a native <video> element with <nui-media-player> to create a custom, skinnable UI.

Try It

Audio Player

Wrap a native <audio> element to render only the audio track controls.

Programmatic Creation

You can dynamically inject a player instance into the DOM using the nui.components.mediaPlayer.create() factory pattern.

Component API

The media player component provides a factory method for programmatic creation.

Factory Methods

Method Description
nui.components.mediaPlayer.create(target, options) Creates a media player and injects it into the target element. Returns the player instance.

Options

Option Description
url The media source URL.
type Media type: "video" or "audio".
poster Poster image URL for video players.
pauseOthers When true (default), automatically pauses other players when this one plays.
attributes Object with attributes to set on the native <video> or <audio> element (e.g., { loop: true, crossorigin: 'anonymous' }).
playerAttributes Object with attributes to set on the <nui-media-player> wrapper element.
