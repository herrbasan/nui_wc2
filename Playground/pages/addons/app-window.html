LLM Guide - nui.appWindow()

App Window

A window chrome module for Electron applications providing a standard title bar, content area, and optional status bar.

Overview

The nui.appWindow() module wraps your application content in a consistent Electron-style frame.

Key features:

  • Title bar with app icon, title, and close button
  • Content area that fills the remaining space
  • Optional status bar at the bottom
  • Right-click context menu on title bar
  • Electron integration (when window.electron_helper is present)
  • Dark mode support via color-scheme
  • Focus/blur state tracking

API Reference

Options

Property Type Description
title string Window title (defaults to document.title)
icon string Icon name from the NUI icon sprite
inner string|HTMLElement Content to display in the main area
statusbar boolean Show status bar (default: false)
target HTMLElement Container to append the app window to (default: document.body)
functions object Custom menu items for title bar context menu
onClose function Callback when close button is clicked

Instance Methods

Method Description
close() Close and remove the app window
setStatusBar(content) Set the status bar text/HTML
center() Center the window (Electron only)
setFullScreen(value) Set fullscreen state (Electron only)
toggleDevTools() Toggle developer tools (Electron only)

Title Bar Context Menu

Right-click on the title bar to access:

  • Toggle Theme - Switch between light and dark mode
  • Custom functions - Any functions you provide in options
  • Toggle DevTools - Open/close DevTools (Electron only)
  • Toggle Fullscreen - Enter/exit fullscreen (Electron only)
  • Center Window - Center on screen (Electron only)
  • Close - Close the window (Electron only)

Dark Mode

The app window uses light-dark() CSS function, automatically responding to the color-scheme property.
