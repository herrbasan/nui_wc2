LLM Guide - nui-lightbox

Lightbox

Dedicated lightbox for image and media galleries.

Declarative Usage

Wrap images in a nui-lightbox to automatically group them. Click on an image to open the lightbox.

Try It

Programmatic Usage

Call the API to show a lightbox without wrapping elements.

Try It

Component API

The lightbox component provides a factory method for programmatic control.

Factory Methods

Method Description
nui.components.lightbox.show(images, startIndex?) Opens the lightbox programmatically. images is an array of objects with src and optional title. startIndex specifies which image to show first (default: 0).

Element Methods

Method Description
open(images, startIndex?) Opens the lightbox on a <nui-lightbox> element. images is an optional array to override the declarative images. startIndex specifies which image to show first.
