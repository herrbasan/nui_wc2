Getting Started with NUI

Build your first NUI application in minutes

Step 1: Include NUI

Add NUI to your HTML file in the <head> section:

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<link rel="stylesheet" href="NUI/css/nui-theme.css">
	<script type="module" src="NUI/nui.js"></script>
</head>
<body>
	<!-- Your content here -->
</body>
</html>

Step 2: Choose Your Layout Mode

NUI supports two layout modes:

App Mode (Desktop Applications)

<nui-app>
	<nui-top-nav>
		<header><h1>My App</h1></header>
	</nui-top-nav>
	<nui-side-nav>
		<nav>Navigation here</nav>
	</nui-side-nav>
	<nui-content>
		<main>Content here</main>
	</nui-content>
</nui-app>

Page Mode (Websites)

<nui-top-nav>
	<header><h1>My Site</h1></header>
</nui-top-nav>
<nui-content>
	<main>Content flows naturally</main>
</nui-content>

Step 3: Add Components

Use NUI components by wrapping semantic HTML:

<nui-button>
	<button type="button">Action</button>
</nui-button>

<nui-link-list mode="tree">
	<ul>
		<li><a href="/home">Home</a></li>
		<li><a href="/about">About</a></li>
	</ul>
</nui-link-list>

Step 4: Add Interactivity

Use the Doer system for actions:

<button nui-event-click="toggle-theme">
	Dark/Light Mode
</button>

<script type="module">
import { nui } from './NUI/nui.js';

nui.registerAction('my-action', (target, source, event, param) => {
	console.log('Custom action executed!');
});
</script>
