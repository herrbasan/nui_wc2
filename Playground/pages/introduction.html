Introduction to NUI

A platform-native UI component library built on web standards

What is NUI?

NUI is a lightweight, performant UI component library that works with the browser platform rather than against it. Built on custom elements and progressive enhancement, it provides a solid foundation for building modern web applications without framework overhead.

Core Principles

  • Platform-Native - Direct browser API usage for maximum performance
  • Semantic HTML First - Components enhance existing markup
  • Progressive Enhancement - Works without JavaScript, better with it
  • Zero Dependencies - Complete control over all code
  • Element Reuse - Cache and reuse DOM elements for performance

Quick Example

<nui-button>
	<button type="button">Click Me</button>
</nui-button>

That's it! The component enhances the native button with NUI behaviors while keeping it accessible.
