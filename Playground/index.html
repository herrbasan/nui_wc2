Testing Modes (URL Parameters)

Use URL parameters to test different scenarios:

  • Normal mode: index.html - Components initialized and enhanced
  • Plain HTML mode: index.html?skip-init - No component initialization, test semantic HTML structure

Plain HTML mode tests progressive enhancement - the page should still be usable without JavaScript.

Test the sidebar navigation functionality:

Active State Management

The link-list component tracks which item is currently active and provides methods to get/set this state.

Current State:

No active item
View API Usage 
// Get the side navigation
const sideNav = document.querySelector('nui-side-nav');

// Set active item (auto-expands parent groups)
sideNav.setActive('a[href="#overview"]');
sideNav.setActive('[data-page="analytics"]');

// Get current active element (returns <li>)
const activeElement = sideNav.getActive();

// Get active data (returns {element, link, href, text})
const data = sideNav.getActiveData();
console.log(data.href);  // "#overview"
console.log(data.text);  // "Overview"

// Clear active state
sideNav.clearActive();

Test Accordion Behavior

Click on any sidebar group header to expand/collapse. With accordion mode enabled, opening one section closes others.
