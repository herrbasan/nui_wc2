Testing Modes (URL Parameters)

Use URL parameters to test different scenarios:

  • Normal mode: index.html - Components initialized and enhanced
  • Plain HTML mode: index.html?skip-init - No component initialization, test semantic HTML structure

Plain HTML mode tests progressive enhancement - the page should still be usable without JavaScript.
